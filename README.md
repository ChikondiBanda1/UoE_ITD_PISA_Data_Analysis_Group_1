# How to use this file

### Method 1 - Download the notebook
1. Download the file: Go to the file list above, click on the notebook (PISA_Code_File_group_1.ipynb), and click the Download raw file button (or the download icon).
<img width="1466" height="615" alt="Screenshot 2025-11-20 at 15 26 32" src="https://github.com/user-attachments/assets/d4cc9b95-b56b-4ad9-9d49-a0c76730ba6c" />

2. Open Notable

3. Upload the .ipynb file you just downloaded.

4. You're in! You should now see the code and text.

NB: Make sure you include the 'GB.csv' dataset in the same folder as PISA_Code_File_group_1.ipynb in Notable otherwise it will not work.
NB: Method 1 is faster than method 2h

### Method 2 - Github Desktop App
1. Download [Github Desktop](https://desktop.github.com/download/).
2. Link Github Desktop app to your github account.
3. From the menu, select File -> Clone Repository
4. In the dialogue that pops up, insert the URL: https://github.com/ChikondiBanda1/UoE_ITD_PISA_Data_Analysis_Group_1
